# Mancora Kite Club

Monorepo with:
- `client/`: React + Vite + SWC + Tailwind v4
- `server/`: Express API + sitemap + SEO-safe trailing slash redirects

## Requirements

- Node.js 20+
- npm 10+

## Install

```bash
npm run install:all
```

## Development

```bash
npm run dev
```

Frontend: `http://localhost:5173`  
Backend: `http://localhost:5000`

## Build

```bash
npm run build
```

## Start server

```bash
npm run start
```

## Lint

```bash
npm run lint
```
